## 📦 你将得到：

一个 `df_final`，包含字段：

| 字段                     | 内容                                   |
| ------------------------ | -------------------------------------- |
| `VolumeName`             | 原始图像名                             |
| `image_path`             | 真实图像路径                           |
| `ClinicalInformation_EN` | 临床信息（如有）                       |
| `Technique_EN`           | 扫描技术描述                           |
| `Findings_EN`            | 图像观察结果                           |
| `Impressions_EN`         | 总结诊断                               |
| `full_report`            | 拼接后的完整报告内容，适合训练生成模型 |
